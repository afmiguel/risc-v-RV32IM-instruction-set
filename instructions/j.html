J - Salto Incondicional (Jump)

Sintaxe

j lab

Descrição

Salta incondicionalmente para o endereço indicado pelo rótulo lab. Esta é uma pseudo-instrução que é traduzida para jal x0, lab.

Formato e Codificação

Formato: Pseudo-instrução (J-type)
Tradução: jal x0, lab
Opcode: 1101111
Alcance: ±1 MB

Operação

PC = PC + offset

Exemplos de Uso

Exemplo 1: Salto para início de loop

loop: addi x10, x10, 1 addi x11, x11, -1 bnez x11, loop # continua loop se x11 != 0 j fim # salta para fim quando loop termina fim: # código continua aqui
Usado para pular incondicionalmente para um rótulo após conclusão de uma tarefa.

Exemplo 2: Salto sobre código condicional

beqz x5, caso_zero # código para x5 != 0 addi x6, x5, 10 j continua # pula o caso_zero caso_zero: # código para x5 == 0 li x6, 0 continua: # código continua aqui
O salto incondicional permite pular blocos de código alternativos após executar um caminho condicional.

Exemplo 3: Loop infinito

inicio: # código principal call funcao_trabalho j inicio # volta sempre para o início
Padrão comum em sistemas embarcados onde o programa deve executar continuamente.

Observações

  • Esta é uma pseudo-instrução expandida para jal x0, lab
  • Não salva endereço de retorno (usa x0 que é sempre zero)
  • O offset é calculado pelo assembler em relação ao PC atual
  • Alcance máximo de ±1 MB (±2^20 bytes) do PC atual
  • Útil para saltos incondicionais sem necessidade de retorno
  • Mais eficiente que usar jal quando não precisa salvar endereço de retorno
