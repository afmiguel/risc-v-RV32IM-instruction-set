LH - Carregar Meia-Palavra (Load Halfword)

Sintaxe

lh rd, imm(rs1)

Descrição

Carrega uma meia-palavra (16 bits) da memória para o registrador de destino rd. O endereço de memória é calculado somando o deslocamento imediato imm ao valor no registrador base rs1. A meia-palavra carregada é então estendida com sinal para 32 bits antes de ser armazenada em rd.

Formato e Codificação

Formato: I-type
Opcode: 0000011
funct3: 0x1

Operação

rd = sign_extend(mem[rs1 + imm])

Exemplos de Uso

Exemplo 1: Carregar de um array de `short`

.data array_shorts: .half 100, -200, 300 .text la t0, array_shorts lh t1, 2(t0) # Carrega o segundo elemento (-200) # O offset é 2 porque cada .half ocupa 2 bytes
`lh` é usado para acessar arrays de inteiros de 16 bits (como `short` em C), preservando seu sinal.

Exemplo 2: Extensão de sinal

# Suponha que o endereço 0x1000 contém a meia-palavra 0x8000 (-32768) li t0, 0x1000 lh t1, 0(t0) # t1 = 0xFFFF8000 (-32768 em 32 bits) lhu t2, 0(t0) # Para comparação, lhu resultaria em 0x00008000 (32768)
A extensão de sinal copia o bit 15 da meia-palavra para os 16 bits superiores de `rd`, mantendo o valor numérico com sinal.

Observações

  • O endereço de memória (rs1 + imm) deve ser alinhado a 2 bytes. Endereços não alinhados podem causar uma exceção de desalinhamento.
  • O deslocamento imm é um valor com sinal de 12 bits (-2048 a +2047).
  • Use `lh` para dados de 16 bits que representam valores com sinal (-32768 a 32767).
  • Para carregar dados de 16 bits sem sinal, use a instrução lhu.
