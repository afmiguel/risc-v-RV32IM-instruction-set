ADDI - Adição com Imediato

Sintaxe

addi rd, rs1, imm

Descrição

Adiciona o valor no registrador rs1 com um valor imediato de 12 bits com extensão de sinal, armazenando o resultado no registrador rd. A operação ignora overflow aritmético.

Formato e Codificação

Formato: I-type
Opcode: 0010011
funct3: 0x0
funct7: N/A

Operação

rd = rs1 + sign_extend(imm[11:0])

Exemplos de Uso

Exemplo 1: Incrementar contador

# Incrementar valor em 1 li x5, 10 # x5 = 10 addi x6, x5, 1 # x6 = 11
Uso mais comum: incrementar um valor por uma constante. Muito usado em loops e contadores.

Exemplo 2: Calcular endereço com offset constante

# Acessar elemento de array la x10, array # x10 = endereço base addi x11, x10, 16 # x11 = endereço do 4º elemento (16 bytes de offset)
Fundamental para acesso a arrays e estruturas. Combina endereço base com offset constante.

Exemplo 3: Subtração com imediato (valor negativo)

# Subtrair constante (não existe SUBI) li x20, 100 # x20 = 100 addi x21, x20, -25 # x21 = 75 (100 - 25)
RISC-V não tem SUBI. Para subtrair constantes, use ADDI com valor negativo. O imediato é estendido com sinal.

Observações

  • O valor imediato é de 12 bits com extensão de sinal (intervalo: -2048 a 2047)
  • ADDI rd, rs, 0 copia rs para rd (pseudo-instrução MV)
  • ADDI rd, zero, imm carrega um valor imediato pequeno em rd (pseudo-instrução LI para valores pequenos)
  • Use valores negativos para implementar subtração de constantes
  • Não gera exceção em caso de overflow
