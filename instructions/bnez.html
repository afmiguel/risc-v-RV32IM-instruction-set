BNEZ - Desvio se Não For Zero (Branch if Not Equal to Zero)

Sintaxe

bnez rs, rotulo

Descrição

Compara o valor no registrador rs com zero. Se não for igual a zero, o fluxo de execução desvia (salta) para o endereço indicado pelo rotulo. Esta é uma pseudo-instrução que o montador traduz para bne rs, zero, rotulo.

Formato e Codificação

Formato: Pseudo-instrução (B-type)
Tradução: bne rs, x0, rotulo

Operação

if (rs != 0) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Loop `do-while`

li t0, 10 # contador loop: # Corpo do loop que executa pelo menos uma vez # ... addi t0, t0, -1 bnez t0, loop # Continua enquanto o contador não for zero
`bnez` é ideal para loops que precisam continuar enquanto um contador for diferente de zero.

Exemplo 2: Checar se um ponteiro não é nulo

# Suponha que a0 contém um ponteiro bnez a0, ponteiro_valido # Salta se o ponteiro não for nulo # ... código para tratar ponteiro nulo ... j fim ponteiro_valido: # ... código para usar o ponteiro ... fim:
Uma verificação de segurança comum antes de desreferenciar um ponteiro para evitar falhas de acesso à memória.

Observações

  • É uma pseudo-instrução muito comum para simplificar o código.
  • O montador a converte em bne rs, x0, rotulo.
  • É o oposto da pseudo-instrução beqz (Branch if Equal to Zero).
