SUB - Subtração

Sintaxe

sub rd, rs1, rs2

Descrição

Subtrai o valor no registrador rs2 do valor no registrador rs1, armazenando o resultado no registrador rd. A operação ignora overflow aritmético.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x0
funct7: 0x20

Operação

rd = rs1 - rs2

Exemplos de Uso

Exemplo 1: Subtração simples

# Subtrair dois números li x5, 100 # x5 = 100 li x6, 42 # x6 = 42 sub x7, x5, x6 # x7 = 58 (100 - 42)
Operação básica de subtração: subtrai o segundo operando do primeiro e armazena o resultado.

Exemplo 2: Calcular diferença entre ponteiros

# Calcular distância entre dois endereços la x10, end_array # x10 = endereço final la x11, start_array # x11 = endereço inicial sub x12, x10, x11 # x12 = tamanho em bytes
Útil para calcular o tamanho de um array ou a diferença entre dois ponteiros em memória.

Exemplo 3: Implementar negação

# Negar um valor (obter -x) li x20, 42 # x20 = 42 sub x21, zero, x20 # x21 = 0 - 42 = -42
Subtrair um valor de zero resulta na sua negação. Esta é a implementação da pseudo-instrução NEG.

Observações

  • Não gera exceção em caso de overflow - o resultado simplesmente "envolve" (wrap around)
  • Funciona tanto para números com sinal quanto sem sinal
  • Não existe SUBI - use ADDI com valor negativo para subtrair constantes
  • SUB rd, rs, rs sempre resulta em zero (x - x = 0)
  • SUB rd, zero, rs nega o valor de rs (0 - x = -x)
