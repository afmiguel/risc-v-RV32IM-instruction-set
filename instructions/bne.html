BNE - Desvio se Não Igual (Branch if Not Equal)

Sintaxe

bne rs1, rs2, rotulo

Descrição

Compara os valores nos registradores rs1 e rs2. Se não forem iguais, o fluxo de execução desvia (salta) para o endereço indicado pelo rotulo. Caso contrário, a execução continua na próxima instrução sequencial.

Formato e Codificação

Formato: B-type
Opcode: 1100011
funct3: 0x1

Operação

if (rs1 != rs2) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Loop com contador decrescente

li t0, 10 # t0 = i = 10 loop: # Corpo do loop addi t0, t0, -1 # i-- bne t0, zero, loop # Continua o loop enquanto i != 0
`bne` é comumente usado para continuar um loop até que um contador chegue a zero.

Exemplo 2: Estrutura if-else

# if (a0 != 5) then ... else ... li t0, 5 beq a0, t0, else_bloco # Se a0 == 5, pula para o bloco else if_bloco: # Código para a0 != 5 j fim_if else_bloco: # Código para a0 == 5 fim_if:
Para implementar um if-else, geralmente se usa a condição de desvio oposta (`beq` neste caso) para pular para o bloco `else`.

Observações

  • O rotulo é convertido pelo montador em um deslocamento (offset) relativo ao PC.
  • O alcance do deslocamento é de ±4 KB.
  • `bne` funciona para operandos com ou sem sinal.
  • A pseudo-instrução bnez rs, rotulo (desvio se não for zero) é implementada como bne rs, zero, rotulo.
