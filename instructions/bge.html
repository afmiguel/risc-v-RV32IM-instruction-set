BGE - Desvio se Maior ou Igual (Branch if Greater or Equal)

Sintaxe

bge rs1, rs2, rotulo

Descrição

Compara os valores nos registradores rs1 e rs2 como inteiros com sinal. Se o valor em rs1 for maior ou igual ao valor em rs2, o fluxo de execução desvia para o endereço indicado pelo rotulo.

Formato e Codificação

Formato: B-type
Opcode: 1100011
funct3: 0x5

Operação

if ((signed)rs1 >= (signed)rs2) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Loop decrescente até zero

li t0, 10 # Inicia o contador t0 com 10 loop: # Corpo do loop addi t0, t0, -1 # Decrementa o contador bge t0, zero, loop # Continua o loop enquanto t0 >= 0
`bge` com o registrador `zero` é uma forma eficaz de fazer um loop enquanto um contador com sinal for não-negativo.

Exemplo 2: Checar um limite mínimo

# if (temperatura >= temp_minima) then ativar_ventoinha li t1, -10 # temp_minima = -10 # Suponha que t0 contém a temperatura atual blt t0, t1, nao_ativa # Pula se temperatura < temp_minima ativar_ventoinha: # ... código para ativar a ventoinha ... nao_ativa:
Para implementar um bloco `if (a >= b)`, a abordagem comum em assembly é usar a condição oposta (`blt`) para pular o bloco.

Observações

  • Realiza uma comparação com sinal. Para comparação sem sinal, use bgeu.
  • A pseudo-instrução bgez rs, rotulo (Branch if Greater or Equal to Zero) é implementada como bge rs, zero, rotulo.
  • É o oposto de blt (Branch if Less Than).
