BEQ - Desvio se Igual (Branch if Equal)

Sintaxe

beq rs1, rs2, rotulo

Descrição

Compara os valores nos registradores rs1 e rs2. Se forem iguais, o fluxo de execução desvia (salta) para o endereço indicado pelo rotulo. Caso contrário, a execução continua na próxima instrução sequencial.

Formato e Codificação

Formato: B-type
Opcode: 1100011
funct3: 0x0

Operação

if (rs1 == rs2) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Loop com contador

li t0, 10 # t0 = contador final li t1, 0 # t1 = i = 0 loop: # Corpo do loop addi t1, t1, 1 # i++ bne t1, t0, loop # Continua se i != 10 # Fim do loop # Usando beq para sair: li t1, 0 loop2: beq t1, t0, fim_loop # Sai do loop se i == 10 # Corpo do loop addi t1, t1, 1 j loop2 fim_loop:
`beq` é frequentemente usado como a condição de término de um loop, saltando para fora do loop quando um contador atinge um valor específico.

Exemplo 2: Estrutura if-then

# if (a0 == a1) then a2 = 1 bne a0, a1, fim_if # Pula o bloco if se a0 != a1 li a2, 1 fim_if:
Em assembly, as condições são frequentemente invertidas. Para executar um bloco `if` quando os valores são iguais, usamos a instrução de desvio oposta (`bne`) para pular o bloco caso a condição seja falsa.

Observações

  • O rotulo é convertido pelo montador em um deslocamento (offset) relativo ao PC (Program Counter).
  • O formato B-type tem um alcance de deslocamento de ±4 KB.
  • `beq` funciona para operandos com ou sem sinal, pois a comparação de igualdade é a mesma.
  • A pseudo-instrução beqz rs, rotulo (desvio se igual a zero) é implementada como beq rs, zero, rotulo.
