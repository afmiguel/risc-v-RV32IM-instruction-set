ANDI - Operação E bit a bit com Imediato

Sintaxe

andi rd, rs1, imm

Descrição

Executa a operação lógica AND bit a bit entre o valor no registrador rs1 e um valor imediato de 12 bits com extensão de sinal, armazenando o resultado no registrador rd.

Formato e Codificação

Formato: I-type
Opcode: 0010011
funct3: 0x7
funct7: N/A

Operação

rd = rs1 & sign_extend(imm[11:0])

Exemplos de Uso

Exemplo 1: Mascaramento de bits inferiores

# Isolar os 8 bits menos significativos li x5, 0xABCD # x5 = 43981 andi x6, x5, 0xFF # x6 = 0xCD = 205 (mantém apenas os 8 bits inferiores)
O ANDI é usado para aplicar uma máscara imediata, isolando os bits desejados. Neste exemplo, apenas os 8 bits menos significativos são preservados.

Exemplo 2: Verificar se número é par

# Verificar se x10 é par andi x11, x10, 1 # x11 = 0 se par, 1 se ímpar
Uma forma compacta de verificar paridade usando ANDI com o valor 1. O resultado será 0 para números pares e 1 para ímpares.

Exemplo 3: Limpar bit específico

# Limpar o bit 3 de um valor li x7, 0x7F7 # máscara: 0111111101110111 (todos os bits em 1, exceto bit 3) andi x8, x9, 0x7F7 # limpa o bit 3 de x9
ANDI é útil para limpar bits específicos quando a máscara cabe em 12 bits. O bit correspondente ao 0 na máscara será zerado no resultado.

Observações

  • O valor imediato é de 12 bits e passa por extensão de sinal para 32 bits
  • Mais eficiente que AND quando se trabalha com valores constantes pequenos
  • Comumente usado para mascaramento e isolamento de bits
  • O imediato pode ser especificado em decimal, hexadecimal ou binário
