XOR - Operação OU-Exclusivo bit a bit

Sintaxe

xor rd, rs1, rs2

Descrição

Executa a operação lógica XOR (OU-Exclusivo) bit a bit entre os valores nos registradores rs1 e rs2, armazenando o resultado no registrador rd.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x4
funct7: 0x00

Operação

rd = rs1 ^ rs2

Exemplos de Uso

Exemplo 1: Inverter bits específicos

# Inverter bits 0 e 3 de um valor li x5, 0b10101010 # x5 = 170 li x6, 0b00001001 # x6 = 9 (máscara para bits 0 e 3) xor x7, x5, x6 # x7 = 0b10100011 (bits invertidos)
XOR com 1 inverte o bit, XOR com 0 mantém o bit. Muito usado para toggle de flags.

Exemplo 2: Zerar um registrador

# Forma eficiente de zerar um registrador xor x10, x10, x10 # x10 = 0 (qualquer valor XOR consigo mesmo = 0)
Esta é uma forma compacta e eficiente de zerar um registrador, usada em otimizações de código.

Exemplo 3: Comparar valores (detectar diferenças)

# Verificar quais bits são diferentes entre dois valores li x20, 0b11001100 li x21, 0b10101010 xor x22, x20, x21 # x22 = 0b01100110 (bits que diferem = 1)
XOR resulta em 1 onde os bits são diferentes e 0 onde são iguais. Útil para detectar mudanças.

Observações

  • XOR resulta em 1 apenas quando os bits são diferentes (um é 0 e outro é 1)
  • XOR resulta em 0 quando os bits são iguais (ambos 0 ou ambos 1)
  • Propriedade importante: A XOR A = 0 (útil para zerar registradores)
  • Usado em criptografia simples, checksums e detecção de erros
  • XOR é reversível: (A XOR B) XOR B = A
