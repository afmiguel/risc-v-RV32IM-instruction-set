SB - Armazenar Byte (Store Byte)

Sintaxe

sb rs2, imm(rs1)

Descrição

Armazena o byte menos significativo (8 bits) do registrador de origem rs2 na memória. O endereço de memória é calculado somando o deslocamento imediato imm ao valor no registrador base rs1.

Formato e Codificação

Formato: S-type
Opcode: 0100011
funct3: 0x0

Operação

mem[rs1 + imm] = rs2[7:0]

Exemplos de Uso

Exemplo 1: Escrever um caractere em uma string

.data minha_string: .space 10 # Aloca 10 bytes .text la t0, minha_string li t1, 'A' # Carrega o código ASCII de 'A' em t1 sb t1, 0(t0) # Escreve 'A' no primeiro byte da string
`sb` é usado para escrever bytes individuais na memória, ideal para modificar strings ou preencher buffers de bytes.

Exemplo 2: Modificar um pixel em uma imagem

# Suponha que t0 contém o endereço de um buffer de imagem # e t1 contém o novo valor de cor (0-255) para um pixel li t2, 100 # Offset do pixel a ser modificado sb t1, 100(t0) # Modifica o 101º byte do buffer da imagem
Em processamento de imagem ou gráficos, `sb` pode ser usado para modificar bytes individuais que representam cores ou outros atributos de pixel.

Observações

  • O deslocamento imm é um valor com sinal de 12 bits, dividido em dois campos na codificação da instrução.
  • Apenas os 8 bits menos significativos de rs2 são escritos na memória.
  • O endereço de memória (rs1 + imm) não precisa estar alinhado.
  • É a instrução fundamental para escrita de dados de 8 bits.
