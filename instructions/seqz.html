SEQZ - Setar se Igual a Zero (Set if Equal to Zero)

Sintaxe

seqz rd, rs

Descrição

Compara o valor no registrador de origem rs com zero. Se o valor for igual a zero, armazena 1 no registrador de destino rd. Caso contrário, armazena 0. Esta é uma pseudo-instrução que o montador traduz para sltiu rd, rs, 1.

Formato e Codificação

Formato: Pseudo-instrução (I-type)
Tradução: sltiu rd, rs, 1

Operação

if (rs == 0) then rd = 1 else rd = 0 endif

Exemplos de Uso

Exemplo 1: Testar o resultado de uma função

jal minha_funcao # Função retorna um código de erro em a0 seqz a1, a0 # a1 = 1 se a0 for 0 (sucesso), senão a1 = 0 (erro) bnez a1, sucesso # Pula se a1 != 0 (ou seja, se a0 era 0)
`seqz` é útil para converter um resultado de zero/não-zero em um valor booleano (0 ou 1), que pode ser facilmente testado.

Exemplo 2: Inverter um valor booleano (0 ou 1)

# Suponha que t0 contém 0 ou 1 seqz t0, t0 # Se t0 era 1, vira 0. Se era 0, vira 1. # Funciona como uma negação lógica (NOT).
Aplicar `seqz` a um valor que só pode ser 0 ou 1 efetivamente inverte esse valor, funcionando como um `not` lógico.

Observações

  • É uma pseudo-instrução, traduzida para sltiu rd, rs, 1. A lógica é que se um número sem sinal `rs` é menor que 1, ele só pode ser 0.
  • O resultado em `rd` é sempre 0 ou 1.
  • É o oposto da pseudo-instrução snez (Set if Not Equal to Zero).
