SW - Armazenar Palavra (Store Word)

Sintaxe

sw rs2, imm(rs1)

Descrição

Armazena a palavra de 32 bits do registrador de origem rs2 na memória. O endereço de memória é calculado somando o deslocamento imediato imm ao valor no registrador base rs1.

Formato e Codificação

Formato: S-type
Opcode: 0100011
funct3: 0x2

Operação

mem[rs1 + imm] = rs2

Exemplos de Uso

Exemplo 1: Salvar um registrador na pilha (push)

# Empilhar o registrador ra antes de uma chamada de função addi sp, sp, -4 # Abre espaço na pilha sw ra, 0(sp) # Salva ra no topo da pilha
Este é um dos usos mais comuns de `sw`: salvar o estado de um registrador na pilha. Essencial para o prólogo de funções que chamam outras funções.

Exemplo 2: Escrever em um array de inteiros

.data meu_array: .space 16 # Aloca 16 bytes para 4 words .text la t0, meu_array li t1, 12345 sw t1, 8(t0) # Escreve 12345 no terceiro elemento do array # (offset de 8 bytes)
`sw` é a instrução padrão para escrever valores de 32 bits em posições de memória, como ao preencher um array de inteiros.

Observações

  • O endereço de memória (rs1 + imm) deve ser alinhado a 4 bytes. Endereços não alinhados podem causar uma exceção.
  • O deslocamento imm é um valor com sinal de 12 bits.
  • É a instrução de armazenamento mais comum para dados de 32 bits.
  • Existe uma pseudo-instrução sw rs, rotulo que é expandida pelo montador para auipc e sw.
