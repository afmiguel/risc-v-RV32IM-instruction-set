SGTZ - Setar se Maior que Zero (Set if Greater Than Zero)

Sintaxe

sgtz rd, rs

Descrição

Compara o valor no registrador de origem rs com zero, tratando-o como um número com sinal. Se o valor for maior que zero, armazena 1 no registrador de destino rd. Caso contrário, armazena 0. Esta é uma pseudo-instrução que o montador traduz para slt rd, zero, rs.

Formato e Codificação

Formato: Pseudo-instrução (R-type)
Tradução: slt rd, zero, rs

Operação

if ((signed)rs > 0) then rd = 1 else rd = 0 endif

Exemplos de Uso

Exemplo 1: Verificar se um número é positivo

# Suponha que a0 contém um número com sinal sgtz t0, a0 # t0 = 1 se a0 > 0, senão t0 = 0 beqz t0, nao_positivo # Pula se não for positivo
`sgtz` fornece uma maneira direta de testar a positividade de um número, convertendo o resultado em um valor booleano.

Exemplo 2: Contar números positivos em um array

# a0 = ponteiro para array, a1 = tamanho, a2 = contador li a2, 0 loop: lw t0, 0(a0) sgtz t1, t0 # t1 = 1 se t0 > 0 add a2, a2, t1 # Adiciona 0 ou 1 ao contador addi a0, a0, 4 addi a1, a1, -1 bnez a1, loop
O resultado de `sgtz` (0 ou 1) pode ser usado diretamente em operações aritméticas para contagem condicional.

Observações

  • É uma pseudo-instrução, traduzida para slt rd, zero, rs. A lógica é que `0 < rs` é o mesmo que `rs > 0`.
  • O resultado em `rd` é sempre 0 ou 1.
  • É o oposto da pseudo-instrução sltz (Set if Less Than Zero).
