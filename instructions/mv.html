MV - Mover Registrador (Move)

Sintaxe

mv rd, rs

Descrição

Copia o valor do registrador de origem rs para o registrador de destino rd. Esta é uma pseudo-instrução que é traduzida para addi rd, rs, 0.

Formato e Codificação

Formato: Pseudo-instrução
Tradução: addi rd, rs, 0

Operação

rd = rs

Exemplos de Uso

Exemplo 1: Passar argumentos para uma função

# Suponha que t0 contém um valor que queremos passar para uma função mv a0, t0 # Copia o valor de t0 para a0 (primeiro argumento) jal minha_funcao
De acordo com a convenção de chamada do RISC-V, os argumentos são passados nos registradores `a0`-`a7`. `mv` é a maneira padrão de mover dados para esses registradores antes de uma chamada de função.

Exemplo 2: Salvar um valor de retorno

jal calcula_valor # Função retorna um valor em a0 mv t1, a0 # Salva o valor de retorno em t1 para uso posterior
Após uma chamada de função, o valor de retorno (em `a0`) é frequentemente movido para um registrador salvo (`s*`) ou temporário (`t*`) para que `a0` fique livre para a próxima chamada.

Observações

  • É uma pseudo-instrução muito comum, traduzida para addi rd, rs, 0.
  • Não há uma instrução de máquina `MV` real no ISA base do RISC-V.
  • É a maneira mais legível e padrão de copiar o conteúdo de um registrador para outro.
