BLTU - Desvio se Menor Que, sem Sinal (Branch if Less Than, Unsigned)

Sintaxe

bltu rs1, rs2, rotulo

Descrição

Compara os valores nos registradores rs1 e rs2 como inteiros sem sinal. Se o valor em rs1 for menor que o valor em rs2, o fluxo de execução desvia para o endereço indicado pelo rotulo.

Formato e Codificação

Formato: B-type
Opcode: 1100011
funct3: 0x6

Operação

if ((unsigned)rs1 < (unsigned)rs2) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Iterar sobre um buffer de memória

la t0, buffer_inicio la t1, buffer_fim loop: # Processa o dado no endereço t0 # ... addi t0, t0, 1 bltu t0, t1, loop # Continua enquanto o ponteiro atual for menor que o final
Como endereços de memória são sempre sem sinal, `bltu` é a instrução correta para verificar se um ponteiro ainda está dentro dos limites de um buffer.

Exemplo 2: Diferença para BLT

li t0, -1 # t0 = 0xFFFFFFFF li t1, 1 # t1 = 0x00000001 blt t0, t1, com_sinal_menor # DEVE saltar (-1 < 1) j continua com_sinal_menor: # Este bloco será executado continua: bltu t0, t1, sem_sinal_menor # Não deve saltar (0xFFFFFFFF > 1) # ...
Este exemplo mostra como `blt` (com sinal) e `bltu` (sem sinal) produzem resultados diferentes para o mesmo padrão de bits quando um deles representa um número negativo.

Observações

  • Realiza uma comparação sem sinal. Para comparação com sinal, use blt.
  • Essencial para aritmética de ponteiros e ao trabalhar com contagens ou tamanhos.
  • É o oposto de bgeu (Branch if Greater or Equal, Unsigned).
