SNEZ - Setar se Não For Zero (Set if Not Equal to Zero)

Sintaxe

snez rd, rs

Descrição

Compara o valor no registrador de origem rs com zero. Se o valor for diferente de zero, armazena 1 no registrador de destino rd. Caso contrário, armazena 0. Esta é uma pseudo-instrução que o montador traduz para sltu rd, zero, rs.

Formato e Codificação

Formato: Pseudo-instrução (R-type)
Tradução: sltu rd, zero, rs

Operação

if (rs != 0) then rd = 1 else rd = 0 endif

Exemplos de Uso

Exemplo 1: Normalizar um valor para booleano

# Suponha que a0 contém um valor qualquer snez t0, a0 # t0 = 1 se a0 for qualquer valor diferente de 0 # t0 = 0 se a0 for 0
`snez` é útil para converter qualquer valor não-zero em 1, e zero em 0, o que é útil para normalizar flags ou valores booleanos.

Exemplo 2: Checar se um ponteiro é válido

# a0 contém um ponteiro snez t0, a0 # t0 = 1 se o ponteiro não for nulo beqz t0, ponteiro_invalido
Testa se um ponteiro é não-nulo, definindo uma flag que pode ser usada para desvio condicional.

Observações

  • É uma pseudo-instrução, traduzida para sltu rd, zero, rs. A lógica é que se `0` sem sinal é menor que `rs` sem sinal, então `rs` não pode ser `0`.
  • O resultado em `rd` é sempre 0 ou 1.
  • É o oposto da pseudo-instrução seqz (Set if Equal to Zero).
