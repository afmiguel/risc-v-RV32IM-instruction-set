BGEU - Desvio se Maior ou Igual, sem Sinal (Branch if Greater or Equal, Unsigned)

Sintaxe

bgeu rs1, rs2, rotulo

Descrição

Compara os valores nos registradores rs1 e rs2 como inteiros sem sinal. Se o valor em rs1 for maior ou igual ao valor em rs2, o fluxo de execução desvia para o endereço indicado pelo rotulo.

Formato e Codificação

Formato: B-type
Opcode: 1100011
funct3: 0x7

Operação

if ((unsigned)rs1 >= (unsigned)rs2) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Comparar ponteiros de memória

# Verificar se ponteiro_atual (t0) ultrapassou o ponteiro_fim (t1) loop: # ... processa dados em t0 ... addi t0, t0, 4 bltu t0, t1, loop # Continua se t0 < t1 (sem sinal) # Usando bgeu: loop2: bgeu t0, t1, fim_loop # Sai se t0 >= t1 # ... processa dados ... addi t0, t0, 4 j loop2 fim_loop:
Endereços de memória são sempre valores sem sinal, então `bgeu` é a instrução correta para compará-los.

Exemplo 2: Diferença para BGE

li t0, -1 # t0 = 0xFFFFFFFF li t1, 1 # t1 = 0x00000001 bge t0, t1, com_sinal_maior # Não deve saltar (-1 < 1) j continua com_sinal_maior: # ... continua: bgeu t0, t1, sem_sinal_maior # DEVE saltar (0xFFFFFFFF > 1) j fim sem_sinal_maior: # Este bloco será executado fim:
Este exemplo mostra como `bge` (com sinal) e `bgeu` (sem sinal) produzem resultados diferentes para o mesmo padrão de bits quando um deles representa um número negativo.

Observações

  • Realiza uma comparação sem sinal. Para comparação com sinal, use bge.
  • Essencial para aritmética de ponteiros e ao trabalhar com dados que são inerentemente sem sinal.
  • É o oposto de bltu (Branch if Less Than, Unsigned).
