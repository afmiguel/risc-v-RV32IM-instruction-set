LW - Carregar Palavra (Load Word)

Sintaxe

lw rd, imm(rs1)

Descrição

Carrega uma palavra (32 bits) da memória para o registrador de destino rd. O endereço de memória é calculado somando o deslocamento imediato imm ao valor no registrador base rs1. Como o registrador já tem 32 bits, nenhuma extensão de sinal ou de zero é necessária.

Formato e Codificação

Formato: I-type
Opcode: 0000011
funct3: 0x2

Operação

rd = mem[rs1 + imm]

Exemplos de Uso

Exemplo 1: Carregar de um array de `int`

.data array_ints: .word 1000, -2000, 3000 .text la t0, array_ints lw t1, 4(t0) # Carrega o segundo elemento (-2000) # O offset é 4 porque cada .word ocupa 4 bytes
`lw` é a instrução padrão para carregar valores de 32 bits da memória, como inteiros de um array ou campos de uma struct.

Exemplo 2: Desreferenciar um ponteiro

.data ponteiro: .word valor valor: .word 0xDEADBEEF .text la t0, ponteiro lw t1, 0(t0) # t1 agora contém o endereço de 'valor' lw t2, 0(t1) # t2 agora contém o dado 0xDEADBEEF
Este exemplo mostra o carregamento de um ponteiro e, em seguida, o uso desse ponteiro para carregar o valor para o qual ele aponta.

Observações

  • O endereço de memória (rs1 + imm) deve ser alinhado a 4 bytes. Endereços não alinhados podem causar uma exceção de desalinhamento.
  • O deslocamento imm é um valor com sinal de 12 bits (-2048 a +2047).
  • É a instrução de carregamento mais comum para dados de 32 bits.
  • Existe uma pseudo-instrução lw rd, rotulo que é expandida pelo montador para auipc e lw.
