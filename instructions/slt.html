SLT - Setar se Menor Que (Set if Less Than)

Sintaxe

slt rd, rs1, rs2

Descrição

Compara os valores nos registradores rs1 e rs2 como inteiros com sinal. Se o valor em rs1 for menor que o valor em rs2, armazena 1 no registrador de destino rd. Caso contrário, armazena 0.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x2
funct7: 0x00

Operação

if ((signed)rs1 < (signed)rs2) then rd = 1 else rd = 0 endif

Exemplos de Uso

Exemplo 1: Comparação de dois números com sinal

li t0, -10 li t1, 10 slt t2, t0, t1 # t2 = 1, porque -10 < 10
`slt` realiza uma comparação aritmética e define o registrador de destino como um booleano (0 ou 1).

Exemplo 2: Implementando `bgt` (Branch if Greater Than)

# bgt rs1, rs2, rotulo (pseudo-instrução) # é implementado como: slt t0, rs2, rs1 # t0 = 1 se rs2 < rs1 (ou seja, rs1 > rs2) bnez t0, rotulo # Desvia se t0 != 0
Muitas pseudo-instruções de desvio condicional são implementadas usando uma instrução `slt` ou `sltu` seguida por um `bnez` ou `beqz`.

Observações

  • Realiza uma comparação com sinal. Para comparação sem sinal, use sltu.
  • O resultado em `rd` é sempre 0 ou 1.
  • É a base para várias pseudo-instruções de comparação e desvio, como `sgt` (Set if Greater Than), `ble` (Branch if Less or Equal), etc.
