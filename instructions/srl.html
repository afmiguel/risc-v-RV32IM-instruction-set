SRL - Deslocamento Lógico à Direita

Sintaxe

srl rd, rs1, rs2

Descrição

Desloca logicamente o valor no registrador rs1 para a direita pelo número de bits especificado nos 5 bits menos significativos de rs2, inserindo zeros nas posições vazias à esquerda, e armazena o resultado em rd.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x5
funct7: 0x00

Operação

rd = (unsigned)rs1 >> (rs2 & 0x1F)

Exemplos de Uso

Exemplo 1: Dividir por potência de 2 (números sem sinal)

# Dividir 80 por 4 (2^2) li x5, 80 # x5 = 80 li x6, 2 # deslocar 2 posições srl x7, x5, x6 # x7 = 20 (80 / 4)
Deslocar à direita por N bits é equivalente a dividir por 2^N para números sem sinal. Neste caso, deslocar por 2 divide por 4.

Exemplo 2: Extrair byte específico

# Extrair o segundo byte (bits 8-15) li x10, 0xABCD1234 # valor de 32 bits li x11, 8 # deslocar 8 bits srl x12, x10, x11 # x12 = 0x00ABCD12 andi x12, x12, 0xFF # x12 = 0x12 (segundo byte)
Combinando SRL com ANDI, podemos extrair bytes específicos de uma palavra. Primeiro deslocamos o byte desejado para a posição inferior, depois isolamos com máscara.

Exemplo 3: Processar campos de bits

# Extrair campo de 4 bits na posição 12 li x20, 0xABCD # valor com campos li x21, 12 # posição do campo srl x22, x20, x21 # desloca campo para LSB andi x22, x22, 0xF # isola os 4 bits (x22 = 0xA)
Útil para decodificar estruturas com múltiplos campos de bits, como instruções ou registradores de controle.

Observações

  • Apenas os 5 bits menos significativos de rs2 são usados (valores de 0 a 31)
  • Zeros são sempre inseridos à esquerda (deslocamento lógico, não aritmético)
  • Bits que saem pela direita são descartados
  • Útil para divisão por potências de 2 de números sem sinal
  • Para números com sinal, use SRA em vez de SRL
