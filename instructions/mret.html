MRET - Retorno de Exceção em Modo M (Machine Return)

Sintaxe

mret

Descrição

Retorna de um manipulador de exceção/interrupção em modo máquina (M-mode). Restaura o PC para o valor salvo em mepc e restaura o estado de interrupções de mpie para mie. Instrução privilegiada que só pode ser executada em modo M.

Formato e Codificação

Formato: I-type
Opcode: 1110011
funct3: 0x0
imm: 0x302

Operação

PC = mepc mstatus.MIE = mstatus.MPIE mstatus.MPIE = 1 # Retorna para modo anterior (se aplicável)

Exemplos de Uso

Exemplo 1: Handler de interrupção básico

trap_handler: # Salvar contexto addi sp, sp, -64 sw x1, 0(sp) sw x5, 4(sp) # ... salva outros registradores # Identificar causa csrr x5, mcause # Processar interrupção/exceção # Restaurar contexto lw x1, 0(sp) lw x5, 4(sp) # ... restaura outros registradores addi sp, sp, 64 mret # retorna da interrupção
Handler completo que salva/restaura contexto e retorna ao ponto interrompido.

Exemplo 2: Handler com análise de causa

interrupt_handler: csrr x6, mcause # lê causa da interrupção bltz x6, eh_interrupcao # É exceção j trata_excecao eh_interrupcao: # Limpa bit de sinal slli x6, x6, 1 srli x6, x6, 1 # x6 agora tem código da interrupção # Processa interrupção específica mret # retorna trata_excecao: # Processa exceção # Pode ajustar mepc se necessário csrr x7, mepc addi x7, x7, 4 # pula instrução que causou exceção csrw mepc, x7 mret # retorna
Handler que diferencia entre interrupções (assíncronas) e exceções (síncronas).

Exemplo 3: Configuração inicial do trap handler

# Configurar vetor de trap la x5, trap_handler csrw mtvec, x5 # define handler # Habilitar interrupções li x6, 0x888 # MIE bits para timer, software, external csrw mie, x6 # Habilitar globalmente li x7, 0x8 # MIE bit csrw mstatus, x7 # Programa continua... # Quando ocorrer interrupção, trap_handler será chamado # e mret retornará para cá
Setup do sistema de interrupções em modo máquina.

Observações

  • Instrução privilegiada - só pode ser executada em modo M (Machine)
  • Restaura PC do registrador mepc (machine exception PC)
  • Restaura estado de interrupção: mstatus.MIE ← mstatus.MPIE
  • Define mstatus.MPIE = 1 (prepara para próxima exceção)
  • Usado no final de handlers de exceção e interrupção
  • Não deve ser confundido com "ret" (retorno de função normal)
  • Handler deve salvar/restaurar registradores usados
  • mepc pode ser modificado antes de mret para controlar retorno
  • Análogo: sret para modo S (Supervisor), uret para modo U (User)
