MULHU - Multiplicar, Parte Alta (Multiply High, Unsigned)

Sintaxe

mulhu rd, rs1, rs2

Descrição

Multiplica os valores nos registradores rs1 e rs2, tratando-os como números sem sinal de 32 bits. O resultado é um valor de 64 bits, e os 32 bits mais significativos (a parte alta) são armazenados no registrador de destino rd.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x3
funct7: 0x01

Operação

rd = ((unsigned)rs1 * (unsigned)rs2) >> 32

Exemplos de Uso

Exemplo 1: Multiplicação de números grandes sem sinal

# Multiplicar 0xFFFFFFFF por 0xFFFFFFFF li t0, -1 # 0xFFFFFFFF li t1, -1 # 0xFFFFFFFF mulhu t2, t0, t1 # t2 = 0xFFFFFFFE (parte alta) mul t3, t0, t1 # t3 = 0x00000001 (parte baixa)
`mulhu` é essencial para aritmética de 64 bits sem sinal, especialmente em criptografia e algoritmos que lidam com grandes números.

Observações

  • Faz parte da extensão "M" do RISC-V.
  • É usada para obter os 32 bits superiores de uma multiplicação de 32x32 bits sem sinal.
  • Para obter os 32 bits inferiores, use a instrução mul.
  • Para operandos com sinal, use mulh.
