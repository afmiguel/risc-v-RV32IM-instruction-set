MULH - Multiplicar, Parte Alta (Multiply High, Signed)

Sintaxe

mulh rd, rs1, rs2

Descrição

Multiplica os valores nos registradores rs1 e rs2, tratando-os como números com sinal de 32 bits. O resultado é um valor de 64 bits, e os 32 bits mais significativos (a parte alta) são armazenados no registrador de destino rd.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x1
funct7: 0x01

Operação

rd = ((signed)rs1 * (signed)rs2) >> 32

Exemplos de Uso

Exemplo 1: Obter resultado completo de 64 bits

# Multiplicar 0x80000000 (-2147483648) por 2 li t0, 0x80000000 li t1, 2 mulh t2, t0, t1 # t2 = 0xFFFFFFFF (parte alta do resultado de 64 bits) mul t3, t0, t1 # t3 = 0x00000000 (parte baixa do resultado) # O resultado de 64 bits é {t2, t3} = 0xFFFFFFFF00000000
Para obter o resultado completo de 64 bits de uma multiplicação, você deve usar `mulh` para a parte alta e `mul` para a parte baixa. Isso é essencial para aritmética de 64 bits em um processador de 32 bits.

Observações

  • Faz parte da extensão "M" do RISC-V.
  • É usada para obter os 32 bits superiores de uma multiplicação de 32x32 bits com sinal.
  • Para obter os 32 bits inferiores, use a instrução mul.
  • Para operandos sem sinal, use mulhu. Para operandos mistos, use mulhsu.
