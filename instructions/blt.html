BLT - Desvio se Menor Que (Branch if Less Than)

Sintaxe

blt rs1, rs2, rotulo

Descrição

Compara os valores nos registradores rs1 e rs2 como inteiros com sinal. Se o valor em rs1 for menor que o valor em rs2, o fluxo de execução desvia para o endereço indicado pelo rotulo.

Formato e Codificação

Formato: B-type
Opcode: 1100011
funct3: 0x4

Operação

if ((signed)rs1 < (signed)rs2) then PC = PC + offset endif

Exemplos de Uso

Exemplo 1: Loop `for` crescente

li t0, 10 # t0 = limite = 10 li t1, 0 # t1 = i = 0 loop: # Corpo do loop addi t1, t1, 1 # i++ blt t1, t0, loop # Continua o loop enquanto i < 10
`blt` é ideal para loops que continuam enquanto um contador é menor que um limite.

Exemplo 2: Checar se um valor é negativo

# if (a0 < 0) then ... blt a0, zero, eh_negativo # ... código para a0 >= 0 ... j continua eh_negativo: # ... código para a0 < 0 ... continua:
Comparar com o registrador `zero` é a forma padrão de testar se um número com sinal é negativo.

Observações

  • Realiza uma comparação com sinal. Para comparação sem sinal, use bltu.
  • A pseudo-instrução bltz rs, rotulo (Branch if Less Than Zero) é implementada como blt rs, zero, rotulo.
  • É o oposto de bge (Branch if Greater or Equal).
