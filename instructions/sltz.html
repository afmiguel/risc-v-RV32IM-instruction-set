SLTZ - Setar se Menor que Zero (Set if Less Than Zero)

Sintaxe

sltz rd, rs

Descrição

Compara o valor no registrador de origem rs com zero, tratando-o como um número com sinal. Se o valor for menor que zero, armazena 1 no registrador de destino rd. Caso contrário, armazena 0. Esta é uma pseudo-instrução que o montador traduz para slt rd, rs, zero.

Formato e Codificação

Formato: Pseudo-instrução (R-type)
Tradução: slt rd, rs, zero

Operação

if ((signed)rs < 0) then rd = 1 else rd = 0 endif

Exemplos de Uso

Exemplo 1: Verificar se um número é negativo

# Suponha que a0 contém um resultado com sinal sltz t0, a0 # t0 = 1 se a0 < 0, senão t0 = 0 bnez t0, eh_negativo # Pula se for negativo
`sltz` é uma maneira explícita de testar se um número é negativo. O bit de sinal de um número com sinal determina se ele é menor que zero.

Exemplo 2: Contar números negativos em um array

# a0 = ponteiro para array, a1 = tamanho, a2 = contador li a2, 0 loop: lw t0, 0(a0) sltz t1, t0 # t1 = 1 se t0 < 0 add a2, a2, t1 # Adiciona 0 ou 1 ao contador addi a0, a0, 4 addi a1, a1, -1 bnez a1, loop
O resultado booleano (0 ou 1) de `sltz` pode ser somado diretamente a um contador.

Observações

  • É uma pseudo-instrução, traduzida para slt rd, rs, zero.
  • O resultado em `rd` é sempre 0 ou 1.
  • É o oposto da pseudo-instrução sgtz (Set if Greater Than Zero), com a exceção do caso de o valor ser zero.
