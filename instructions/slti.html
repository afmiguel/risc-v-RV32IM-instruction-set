SLTI - Setar se Menor Que, Imediato (Set if Less Than, Immediate)

Sintaxe

slti rd, rs1, imediato

Descrição

Compara o valor no registrador rs1 com o valor imediato, tratando ambos como números com sinal. Se rs1 for menor que o imediato, armazena 1 no registrador de destino rd. Caso contrário, armazena 0.

Formato e Codificação

Formato: I-type
Opcode: 0010011
funct3: 0x2

Operação

if ((signed)rs1 < (signed)imediato) then rd = 1 else rd = 0 endif

Exemplos de Uso

Exemplo 1: Verificar um limite superior

# Suponha que t0 contém uma leitura de sensor slti t1, t0, 100 # t1 = 1 se t0 < 100, senão t1 = 0 beqz t1, valor_ok # Pula se t1 for 0 (t0 >= 100) # ... código para lidar com valor abaixo do limite ... valor_ok:
`slti` é útil para comparações diretas com constantes, sem precisar carregar a constante em outro registrador.

Exemplo 2: Comparação com número negativo

li t0, -50 slti t1, t0, -40 # t1 = 1, porque -50 < -40
O valor imediato é estendido com sinal, permitindo comparações corretas com números negativos.

Observações

  • O valor imediato é um número com sinal de 12 bits (faixa de -2048 a +2047).
  • Realiza uma comparação com sinal. Para comparação sem sinal com um imediato, use sltiu.
  • O resultado em `rd` é sempre 0 ou 1.
