CSRC - Limpar Bits de CSR (Clear bits in CSR)

Sintaxe

csrc csr, rs

Descrição

Limpa bits específicos do registrador de controle e status csr usando o conteúdo do registrador rs como máscara de bits. Os bits que são 1 em rs são zerados no CSR. Esta é uma pseudo-instrução traduzida para csrrc x0, csr, rs.

Formato e Codificação

Formato: Pseudo-instrução (I-type)
Tradução: csrrc x0, csr, rs
Opcode: 1110011
funct3: 0x3

Operação

csr = csr & ~rs

Exemplos de Uso

Exemplo 1: Desabilitar interrupções específicas

# Desabilitar interrupções de timer e externa li x5, 0x880 # bits para timer (11) e externa (7) csrc mie, x5 # limpa esses bits em mie # Agora timer e externa estão desabilitadas
Usado para desabilitar interrupções específicas sem afetar outras configurações.

Exemplo 2: Desabilitar interrupções globais

# Seção crítica - desabilita interrupções li x5, 0x8 # bit MIE (Machine Interrupt Enable) csrc mstatus, x5 # desabilita interrupções globais # Código da seção crítica # ... # Reabilita interrupções csrs mstatus, x5
Padrão comum para proteger seções críticas do código contra interrupções.

Exemplo 3: Limpar múltiplos bits de status

# Limpar bits MPP (Machine Previous Privilege) li x6, 0x1800 # bits 11 e 12 (MPP field) csrc mstatus, x6 # limpa campo MPP # Agora MPP = 00 (User mode)
Útil para configurar campos multi-bit em registradores de status.

Observações

  • Esta é uma pseudo-instrução expandida para csrrc x0, csr, rs
  • Não lê o valor antigo do CSR (descarta em x0)
  • Operação atômica: read-modify-write em um ciclo
  • Bits que são 1 em rs são zerados no CSR
  • Bits que são 0 em rs permanecem inalterados no CSR
  • Útil para desabilitar flags, interrupções ou recursos específicos
  • Complementar à instrução csrs (que seta bits)
  • Acesso a CSRs requer privilégios adequados
  • Se rs = x0, a operação não modifica o CSR
