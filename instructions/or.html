OR - Operação OU bit a bit

Sintaxe

or rd, rs1, rs2

Descrição

Executa a operação lógica OR bit a bit entre os valores nos registradores rs1 e rs2, armazenando o resultado no registrador rd.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x6
funct7: 0x00

Operação

rd = rs1 | rs2

Exemplos de Uso

Exemplo 1: Combinar flags de status

# Combinar múltiplas flags de status li x5, 0b00000001 # x5 = flag de erro (bit 0) li x6, 0b00000100 # x6 = flag de aviso (bit 2) or x7, x5, x6 # x7 = 0b00000101 (ambas as flags ativas)
O OR é usado para combinar múltiplas flags ou condições, preservando todos os bits que estão em 1 em qualquer dos operandos.

Exemplo 2: Ativar bits específicos

# Ativar bits 3 e 5 em um registrador li x10, 0b01010000 # valor original li x11, 0b00101000 # máscara com bits 3 e 5 or x12, x10, x11 # x12 = 0b01111000 (bits ativados)
Útil para configurar bits específicos sem afetar os outros bits do registrador.

Exemplo 3: Verificar se algum bit está ativo

# Verificar se qualquer flag de erro está ativa li x20, 0xFF00 # máscara para bits de erro (8-15) or x21, x20, x22 # combina máscara com registrador de status # Se x21 != 0, algum erro está ativo
O OR pode ser usado para detectar se algum de vários bits está ativo.

Observações

  • A operação OR resulta em 1 se pelo menos um dos bits correspondentes for 1
  • Muito usado para ativar flags ou combinar máscaras de bits
  • O resultado de um bit é 0 apenas quando ambos os bits correspondentes são 0
  • Útil para configurar registradores de controle em programação de hardware
