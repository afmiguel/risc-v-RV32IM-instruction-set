ADD - Adição

Sintaxe

add rd, rs1, rs2

Descrição

Adiciona os valores nos registradores rs1 e rs2, armazenando o resultado no registrador rd. A operação ignora overflow aritmético.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x0
funct7: 0x00

Operação

rd = rs1 + rs2

Exemplos de Uso

Exemplo 1: Soma simples

# Somar dois números li x5, 42 # x5 = 42 li x6, 58 # x6 = 58 add x7, x5, x6 # x7 = 100
A operação mais básica: adicionar dois valores e armazenar o resultado em um terceiro registrador.

Exemplo 2: Calcular endereço com offset

# Calcular endereço: base + offset la x10, array # x10 = endereço base do array li x11, 20 # x11 = offset de 20 bytes add x12, x10, x11 # x12 = endereço efetivo (base + 20)
Comum em operações de acesso à memória, onde combinamos um endereço base com um deslocamento calculado.

Exemplo 3: Acumular valores em loop

# Acumular soma li x20, 0 # acumulador = 0 li x21, 10 # valor a adicionar add x20, x20, x21 # acumulador += 10 add x20, x20, x21 # acumulador += 10 (agora = 20)
Padrão comum em loops onde acumulamos valores. Note que rd e rs1 podem ser o mesmo registrador.

Observações

  • Não gera exceção em caso de overflow - o resultado simplesmente "envolve" (wrap around)
  • Funciona tanto para números com sinal quanto sem sinal
  • Para adicionar constantes, use ADDI que é mais eficiente
  • rd, rs1 e rs2 podem ser o mesmo registrador (ex: add x5, x5, x5 dobra o valor)
  • A instrução mais fundamental em praticamente todos os programas RISC-V
