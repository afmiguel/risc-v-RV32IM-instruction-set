AND - Operação E bit a bit

Sintaxe

and rd, rs1, rs2

Descrição

Executa a operação lógica AND bit a bit entre os valores nos registradores rs1 e rs2, armazenando o resultado no registrador rd.

Formato e Codificação

Formato: R-type
Opcode: 0110011
funct3: 0x7
funct7: 0x00

Operação

rd = rs1 & rs2

Exemplos de Uso

Exemplo 1: Mascaramento de bits

# Isolar os 4 bits menos significativos li x5, 0b11010110 # x5 = 214 (binário: 11010110) li x6, 0b00001111 # x6 = 15 (máscara: 00001111) and x7, x5, x6 # x7 = 6 (binário: 00000110)
O AND é usado para aplicar uma máscara, mantendo apenas os bits desejados. No exemplo acima, apenas os 4 bits menos significativos de x5 são preservados.

Exemplo 2: Verificar se número é par

# Verificar se x10 é par (bit 0 == 0) li x11, 1 # máscara para bit menos significativo and x12, x10, x11 # x12 = 0 se par, 1 se ímpar
Se o resultado for 0, o número é par. Se for 1, é ímpar. Isso funciona porque números pares têm o bit menos significativo igual a 0.

Exemplo 3: Limpar bits específicos

# Limpar bits 4-7 de um registrador li x20, 0xFF0F # máscara: 1111111100001111 and x21, x20, x22 # limpa bits 4-7 de x22
Útil para manipulação de campos de bits em estruturas de dados. Os bits que são 0 na máscara serão zerados no resultado.

Observações

  • A operação AND preserva apenas os bits que são 1 em ambos os operandos
  • Muito usado para mascaramento e isolamento de bits específicos
  • O resultado de um bit é 1 apenas quando ambos os bits correspondentes são 1
  • Útil para verificar flags e condições específicas em programação de baixo nível
